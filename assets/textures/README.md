# Texture Pipeline

Drop generated texture files here. The game will use these files when present and fall back to built-in procedural textures while they are missing.

Suggested filenames:

- `asphalt_basecolor.jpg`
- `sidewalk_basecolor.jpg`
- `grass_basecolor.jpg`
- `house_siding_basecolor.jpg`
- `house_siding_red_basecolor.jpg`
- `roof_shingles_basecolor.jpg`
- `wood_floor_basecolor.jpg`
- `drywall_basecolor.jpg`
- `car_paint_basecolor.jpg`
- `fabric_basecolor.jpg`
